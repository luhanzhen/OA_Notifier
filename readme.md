
# 😀 😁 This is a rust project for the OA notification.

>每十分钟读取[OA](https://oa.jlu.edu.cn/defaultroot/login.jsp?access=oa)上的前三页，也就是前90条通知，如果有新消息，会通知的。
>虽然是爬虫，但是不会给OA服务器带来巨额的压力。
>注意一定要在能访问OA的环境下使用。
